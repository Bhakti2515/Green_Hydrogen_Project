# Design and Analysis of a Multi-port DC-DC Converter for Green Hydrogen Production

## Project Overview
This project presents the design and simulation of a Multi-port DC-DC Converter for Green Hydrogen Production using MATLAB/Simulink. The system integrates multiple DC energy sources, including a solar DC source and battery backup, to provide stable and regulated DC output for hydrogen electrolyzer applications.

## Working Principle
The converter combines power from solar and battery sources using a multi-port DC-DC converter topology. The MOSFET operates as a high-speed switching device controlled by PWM pulses generated through the pulse generator block.

The inductor stores and transfers energy, while the capacitor smooths the output voltage and reduces ripple. The diode ensures proper current flow direction, and the resistor acts as the electrolyzer load.

The output voltage is monitored using a voltage sensor and analyzed through waveform visualization in Simulink Scope.

---

## Simulation Results
The simulation output waveform demonstrates transient response followed by stable steady-state converter operation, indicating effective voltage regulation and proper converter functionality.
