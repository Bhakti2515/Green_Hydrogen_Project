# Design and Analysis of a Multi-port DC-DC Converter for Green Hydrogen Production

## Project Overview
This project presents the design and simulation of a multi-port DC-DC converter for green hydrogen production using MATLAB and Simulink.

The system integrates:
- Solar DC Source
- Battery Backup Source
- DC-DC Converter
- Electrolyzer Load

The converter provides stable DC output suitable for renewable energy and hydrogen production applications.
